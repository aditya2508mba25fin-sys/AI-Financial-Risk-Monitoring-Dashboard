# 🏦 NeoFin AI – Financial Risk Monitoring & Fraud Detection System

An AI-powered Financial Risk Monitoring and Fraud Detection platform built using **n8n, AI Agents, Rule-Based Automation, Google Sheets, and Real-Time Alerting**. The system continuously monitors transactions, detects suspicious activities, scores risk levels, and automatically escalates high-risk cases for investigation.

---

## 📌 Project Overview

Financial institutions process thousands of transactions daily, making manual monitoring inefficient and error-prone.

**NeoFin AI** automates fraud detection by combining:

- Rule-Based Risk Assessment
- AI-Powered Analysis
- Real-Time Transaction Monitoring
- Automated Alerts & Escalations
- Executive-Level Dashboards

The platform identifies potentially fraudulent transactions and provides actionable insights for risk management teams.

## 📈 Risk Detection Logic

The system evaluates transactions against multiple fraud indicators:

### High Transaction Amount
- Unusually large transactions
- Threshold-based monitoring

### Location Anomaly
- Transaction initiated from unfamiliar locations
- Geo-risk analysis

### Suspicious Login Activity
- Multiple failed login attempts
- Unrecognized device detection

### Transaction Velocity
- Multiple transactions within short periods
- Rapid account activity monitoring

---

## 🔄 Workflow Process

1. Transaction enters the system.
2. Risk engine evaluates fraud rules.
3. AI agent performs contextual analysis.
4. Risk score is generated.
5. High-risk transactions are flagged.
6. Alerts are sent automatically.
7. Executive dashboard updates in real time.
